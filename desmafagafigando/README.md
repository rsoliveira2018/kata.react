# DESMAFAGAFIGANDO

A project for showcasing the adventures of a group of friends in outdoors activities, including cycling, walking, running, hiking, camping, and possibly others.

# technologies

The main technologies employed here were:
- Docker
- MySQL
- .NetCore
- EntityFramework
- React
- Bootstrap
